# Metamask Login
Project for testing the Metamask API 

## Technologies
- node 14.18.0
- Next.js 
- Tailwind CSS

## Installation
1. Install packages 
    ```bash
    npm install
    ```

2. Start the project
   ```bash
   npm run dev
   ```

3. Build the project (Production)
   ```bash
   npm run build
   ```
4. Start the project on Production (Production)
   ```bash
   npm start
   ```
